Getting Started with the Responsive Classic Redesign Template

  1. Clone or Download the Project: Clone or download the demo files from this repository to your local machine.
  2. Upload Files to Your Server: Extract the downloaded files and upload the 'images' and 'includes' folder to your web server.
    - Optional: You can also download the 'updated_demo_image.zip' for additional image assets.
    - Optional: Add the CSS file when using the One Page Checkout plugin to match site styling.
  3. Activate the Template: Log in to Zen Cart and navigate to Tools > Template Section.
  4. Activate the responsive_classic template.

Making Changes to Template

Columns per Row – What It Does

Description: The Admin Columns per Row setting controls how many items are shown in each row in the product or category view.

Location: Admin > Configuration > Product Listing > Columns per Row

Note: Any change to the column count must be reflected in your CSS to maintain proper layout alignment. The default template is already optimized for 1 or 3 columns, so no additional CSS changes are required.

Example CSS

There are two layout styles: one for mobile and one for desktop.
Make sure the desktop column count matches your “Columns per Row” setting in the Zen Cart admin.

path includes/templates/responsive_classic/css/stylesheet_zcustom.css

/* Mobile view – 1 product per row */
    .rblockelements {
    display: grid;
    grid-template-columns: repeat(1, 1fr);
}

/* Desktop view – match this number to your admin setting */
.rblockelements {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Change 3 to match Columns per Row setting */
    gap: 1rem;
}

Change Position of Navigation Items

The top navbar has a class .nav_spacer which is a flex container. You can use the order style to move items within that container.

To move the hamburger menu to the right:

.nav_spacer .row {
    order: 5;
}

To move the search menu to the right:

.nav_spacer .search_box {
    order: 4;
}

Display the Sidebar on the Right Side

This guide explains how to apply custom styling for OPC Checkout to match the website theme.

OPC must be installed before moving forward ***


Steps to Install

1.  Move the custom stylesheet to the following path:
    	includes/templates/responsive_classic/css/stylesheet_zcustom_opc.css

2.  - optional - upload the includes directory to the website root directory, preserving the existing folder structure.  
   (Changes in this folder is to ensure we are in compliance with Web Content Accessibility Guidelines)
